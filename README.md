## üì¶ Installation

### Chrome Web Store (Recommended)

1. Visit the [Minimalist Orange Juice](https://chromewebstore.google.com/detail/minimalist-orange-juice/cphbblmnejdobpelaamimglcmjkcccnj) theme page on the Chrome Web Store.
2. Click the **"Add to Chrome"** button to install the theme directly from the store.

### From Source

1. Clone this repository.
2. Open `chrome://extensions` and enable Developer mode.
3. Click "Load unpacked" and select the theme folder.
